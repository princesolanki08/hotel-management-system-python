# hotel-management-system-python
A Python-based Hotel Management System built using Tkinter for GUI, designed to manage customer records, room bookings, and billing operations efficiently. This project runs locally and is not deployed due to its desktop-based architecture and dependency on system libraries.


## ⚠️ Note on Deployment

This project is a desktop-based application built using Python and Tkinter.  
It cannot be deployed on web hosting platforms like Vercel or Netlify.

To run this project, you need:

- Python installed on your system
- Required libraries (Tkinter, Pillow, etc.)

Run locally using:

```bash
python main.py
